# 3DViewer

Программа 3DViewer представляет собой приложение для просмотра и визуализации трехмерных моделей. Она обладает следующими возможностями:

1. Установка
   Для установки приложения выполните команду `make install`. Это создаст необходимые директории, скомпилирует программу, переместит исполняемый файл в папку `bin` и откроет приложение.

2. Запуск
   Вы можете запустить приложение командой `make run`. Если приложение еще не установлено, вас спросят, хотите ли вы его установить. Вы также можете использовать команду `make install`, чтобы установить и запустить приложение.

3. Архивация
   Вы можете упаковать программу в тар-архив с помощью команды `make dist`.

4. Возможности программы
   - Приложение поддерживает сохранение и загрузку сохраненных файлов, включая настройки цвета фона, цвета модели, цвета вершин и их типов, расположения, ротации и размера модели.
   - Программа работает с файлами .obj, где количество вершин и линий не превышает 3 позиции.
   - Возможно записывать действия пользователя в формате .gif. Также есть возможность создания скриншотов.
   - Реализовано отображение пунктирных линий, текстур, вершин (круги/квадраты) и изменение цвета модели, вершин и фона.
   - Возможно изменение ротации, размеров и перемещение модели. Реализована возможность вращения модели при помощи клавиш WASD и управления мышью.
   - Реализованы два типа камеры для отображения модели.
   - Присутствует возможность сохранения текущего состояния программы.

---

**Примечание:** Для успешного использования приложения, убедитесь, что у вас установлены следующие зависимости:

- QtQuick3D (Qt 5)
- QtQuick3D (Qt 6)
- CMake
- make

Инструкции по установке зависимостей доступны в соответствующей документации для вашей операционной системы.
